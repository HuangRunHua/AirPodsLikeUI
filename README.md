# AirPodsLikeUI
这是一个参考AirPods连接界面制作的demo。

## 环境要求
- Xcode9.0 及以上
- iOS12.0 及以上
- swift4.0 及以上

## 功能示例

下面一张图片展示了这个demo的主要功能，，功能比较简单，用户点击加号按钮以后可以调出一个新的界面，通过修改选择器的值来改变模糊效果
